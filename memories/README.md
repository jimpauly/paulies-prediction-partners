# Memories

Short, journal-style notes for major decisions or direction changes.

When to add a note:
- Major decision
- Direction change
- User explicitly says "I love that"

Format:
- One file per decision in `memories/`
- Filename: `YYYY-MM-DD--short-title.md`
- 1-3 lines, plain text

Example:
`2026-03-13--quadrant-ii-scale.md`
- Kept 1/4 scale for the tiny cockpit layout.
